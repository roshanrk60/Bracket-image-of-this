# Mission 3 — Mechanical Design Project
**Name:** Roshan Koreti

## Project: Adjustable Phone/Tablet Stand

## Description
A 3-part mechanical assembly designed in Autodesk Fusion,
consisting of a base plate, vertical support arm, and
adjustable bracket — simulating a real product design workflow.

## Parts Designed
- Base Plate: 120mm x 80mm x 10mm with mounting holes
- Vertical Arm: 20mm x 20mm x 150mm support column
- Bracket: 80mm x 60mm adjustable holder with slots

## Tools Used
- Autodesk Fusion (CAD + Assembly + Render)
- GitHub (Documentation)

- Multi-part assembly with joints
